# SEI Sports Site

A fast frontend built with **Vite + React + TailwindCSS** for SEI Sports Fans.

## 📦 Setup

```bash
npm install
npm run dev
```

## 💡 Features

- ⚡ Vite-powered speed
- 🎨 TailwindCSS for styling
- 🚀 Ready to expand with components/pages

## 🛠️ Folder Structure

```
src/
├── components/
├── pages/
│   └── Home.jsx
└── main.jsx
```
